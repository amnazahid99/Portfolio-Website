## Features

- Add Task: Create a new task with a description
- Remove Task: Delete a task by its index.
- Toggle Task: Change the completion status of a task.
- Update Task: Modify the description of an existing task.
- Get Tasks: Retrieve the current list of tasks.

## Usage/Examples

Here's how you can utilize the functions provided by this package:

### Add a Task

```javascript
const { addTask } = require('your-package-name');

addTask('Read a book');
```
Description: Adds a new task with the given description.


### Remove a Task

```javascript
const { removeTask } = require('your-package-name');

removeTask(1); // Removes the first task

```
Description: Removes the task at the specified 1-based index.

### Toggle Task Completion

```javascript
const { toggleTask } = require('your-package-name');

toggleTask(1); // Toggles completion status of the first task
```
Description: Changes the completion status of the task at the specified 1-based index.

### Update a Task

```javascript
const { updateTask } = require('your-package-name');

updateTask(1, 'Write a blog post'); // Updates the description of the first task

```
Description: Updates the task description at the specified 1-based index.

### Get all Tasks

```javascript
const { getTasks } = require('your-package-name');

console.log(getTasks()); // Logs the current list of tasks
```
Description: Retrieves and displays the current list of tasks.

## API Reference

Add Task

```javascript
addTask(description)
```
| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| 'description'     | string | **Required**. Description of task |

Remove Task

```javascript
removeTask(taskIndex)

```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| 'taskIndex'     | 'number'	 | **Required**. Index of task to remove |

Toggle Task

```javascript
toggleTask(taskIndex)

```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| 'taskIndex'     | 'number'	 | **Required**. Index of toggle status |

Update Task

```javascript
updateTask(taskIndex, newDescription)
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| 'taskIndex'     | 'number'	 | **Required**. Index of task to update |
| 'newDescription'     | 'string'	 | **Required**. New description of task|

getTasks()
